# Safe Clear

Save session context and extract reusable learnings before clearing context.

## Workflow
1. Read the required docs listed in `references/original-command.md` before extracting learnings.
2. Scale the depth of analysis to the session size.
3. Update only the designated project memory files, never external memory systems.
4. End with the exact clear-ready phrasing from the legacy command.
