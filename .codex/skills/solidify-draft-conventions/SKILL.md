# Solidify Draft Conventions

Promote or prune staged conventions one item at a time.

## Workflow
1. Read `docs/conventions_draft.md`, `docs/conventions.md`, and `docs/personal_preferences.md` before starting.
2. Present one draft entry at a time and explicitly state whether it is duplicate or a candidate for generalization.
3. Prefer widening an existing convention over adding a narrow new one.
4. Finish with the exact cleanup and preference-extraction steps from `references/original-command.md`.

## References
- `references/original-command.md` -> review workflow and final cleanup rules
- `docs/conventions_draft.md` -> staged convention candidates
- `docs/conventions.md` -> permanent convention set
