# Examples

These calibrate Cultural status / canonical stature as whole-work reputation, not specific aspect praise.

**Example: "cult classics"**

SEMANTIC should fire with reception prose about a devoted following and lasting subcultural reputation. METADATA should usually stay silent because "cult" is a reception shape, not a simple quality or popularity direction.

**Example: "modern classics"**

The cultural-status atom should fire SEMANTIC for canonical stature and METADATA as a supporting quality/popularity prior. The "modern" era atom belongs to a separate Release date / era handler call; do not encode the date window here.

**Example: "underrated thrillers"**

SEMANTIC carries the quality-versus-recognition gap. METADATA should not turn this into a hard `well_received` requirement; if it fires at all, explain that the scalar is only a weak supporting prior and cannot express "better than its reputation suggests".
