# Additional objective notes

## Category Target

Named cinema tradition, movement, or national-cinema aesthetic:
Bollywood, Korean cinema, Hong Kong action, French New Wave, Dogme 95.
Ask: "Is this a cinema tradition, not just a country or language?"

## Coverage Decision

- Preferred: Keyword when a registry member directly names the tradition
  or movement.
- Fallback: Metadata country/language only when no tradition tag covers
  the ask.
- Do not run Metadata when Keyword covers the tradition. Country is a
  lossy proxy and can miss diaspora, co-production, and aesthetic cases.

## Boundaries

- Legal/financial production country -> Country of origin.
- Original audio language -> Audio language.
- Physical shooting location -> Filming location.
- Genre without tradition framing -> Genre.

## No-Fire

No-fire for vague "foreign feel", explicit country paperwork, explicit
language, or location-only wording.
