# Additional objective notes

## Category Target

Overarching story shape, theme, or character-arc trajectory: grief,
redemption, found family, coming-of-age, man-vs-nature, revenge,
underdog, survival, moral compromise. Ask: "What kind of story is it?"

## Coverage Decision

- Preferred: Keyword when a registry member directly covers a
  binary-framed story shape or theme.
- Fallback: Semantic plot_analysis when no tag covers the concept.
- Spectrum rule: if the wording is graded ("kind of", "leans", "hint
  of", "thread of"), use Semantic even if a tag exists. Binary tags are
  the wrong tool for degree.
- Split: use Keyword for the covered binary shape and Semantic only for
  uncovered thematic qualifiers inside this category call.

## Boundaries

- Concrete focal subject -> Central topic.
- Static character type -> Character archetype.
- Genre identity -> Genre.
- Literal event sequence -> Plot events.

## No-Fire

No-fire for vague depth language, static character labels, concrete
subjects, or genre/form requests.
