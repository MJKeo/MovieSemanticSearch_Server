# Additional objective notes - Chronological ordinal

## Target

Fire for release-date rank within the scoped candidate set: first, last,
earliest, latest, most recent, newest, oldest.

## Decision Questions

- Does the phrase select by position in release-date order?
- What candidate set is scoped by sibling traits?
- Which direction is implied: earliest/first or latest/last?

## Boundaries

- "Most recent" is ordinal. "Recent" is a release-date range.
- "First Bond" selects by chronology; "best Bond" selects by reception.
- Never approximate an ordinal as a date range.
- Do not emit sibling scope such as director/franchise/title here; siblings
  define the pool.
