# Additional objective notes

## Category Target

Static character type or role pattern: anti-hero, femme fatale, lovable
rogue, reluctant hero, love-to-hate villain. Ask: "What type of
character must be present?"

## Coverage Decision

- Preferred: Keyword when a registry member directly defines the
  archetype.
- Fallback: Semantic narrative_techniques when the archetype is real
  but untagged.
- Split: use Keyword for the covered type and Semantic only for an
  uncovered subtype or qualifier.

Do not pick an adjacent tag to satisfy the keyword preference. A broad
character tag that misses the flavor is not coverage.

## Boundaries

- Specific named persona -> Named character or Character-franchise.
- Character trajectory or theme -> Story / thematic archetype.
- Ensemble/single-lead structural configuration -> Narrative devices.
- Mere story element -> Element / motif presence.

## No-Fire

No-fire for vague character quality, named character, arc trajectory,
or structural cast framing.
