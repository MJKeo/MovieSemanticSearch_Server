# Additional objective notes

## Category Target

Genre identity: broad genres and named subgenres. Ask: "What genre or
subgenre family should this belong to?"

## Coverage Decision

- Preferred: Keyword when a registry member directly names the genre or
  subgenre.
- Fallback: Semantic plot_analysis.genre_signatures when the genre
  texture is untagged, qualified, hybrid, or too specific for the
  registry.
- Split: use Keyword for a covered base genre and Semantic for an
  uncovered qualifier only when the qualifier stays inside this category
  call.

Do not use a parent genre tag for a named subgenre it does not cover.
Do not split recognized subgenre compounds merely because they contain
a broad genre word.

## Boundaries

- Story shapes like revenge, underdog, survival, post-apocalyptic ->
  Story / thematic archetype.
- Formats like documentary or anime -> Format + visual-format specifics.
- Pure tone without genre anchor -> Emotional / experiential.

## No-Fire

No-fire for vague genre language, pure mood, format, story archetype,
or any expression where neither tag nor genre_signatures can target a
real genre identity.
