# Additional objective notes - Visual craft acclaim

Search praise or notable craft around the image: cinematography, production
design, visual effects, practical effects, composition, lighting, visual
spectacle, or technical visual achievement.

Use semantic reception when the request is acclaim/praise ("praised for
cinematography", "visually stunning"). Use semantic production when the
request names physical technique ("practical effects", "shot on IMAX",
"miniatures"). Use both only when both signals are explicit.

Do not turn named craft people into this category. Literal
cinematographer/editor/VFX-supervisor credit asks are reserved elsewhere;
style-transfer asks should already have been resolved into concrete visual
attributes upstream.

No-fire for plain genre/format, vague beauty with no craft target, or named
person credit phrasing.
