# Additional objective notes - Release date / era

## Target

Fire for release-date windows: exact years, decades, before/after ranges,
named eras, and vague era words.

## Decision Questions

- Does the phrase name a date window, not a rank? If yes, use release date.
- Is the date vague ("recent", "modern", "old-school")? Resolve the
  implied window; do not turn it into an ordinal.
- Does the phrase mean live buzz ("trending now")? No-fire; that is not a
  release-date window.

## Boundaries

- "Recent" = range. "Most recent" = ordinal.
- "Classic" alone = cultural status, not era. "Old classic" / "modern
  classic" can have a separate release-date slice.
- Do not use film knowledge to infer unstated eras.
