# Additional objective notes - Narrative setting

Search where or when the story takes place. Use semantic
`plot_events.plot_summary`, but phrase as setting description:
"set in...", "takes place in...", "during...".

This is narrative time/place, not release date, filming location, production
country, or national cinema tradition. "Set in 1940s Berlin" belongs here;
"released in the 1940s" and "filmed in Berlin" do not.

Keep the body descriptive and compact. If the request gives only setting,
do not invent a plot around it. A setting-only synopsis fragment is valid.

No-fire when the expression names physical shooting geography, real-world
release era, production origin, or a vague atmosphere with no place/time.
