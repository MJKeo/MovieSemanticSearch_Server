# Additional objective notes - Named source creator

Search for a named creator of source material: author, playwright, comics
creator, or writer whose work the film adapts. Use semantic plot/reception
text where matching movies mention the source creator.

The name itself stays here; the medium flag is separate. "Stephen King
novels" should fire this category for Stephen King and Adaptation source
for novel.

Do not confuse source creators with film credits. "Directed by Stephen
King" or "screenplay by Jane Goldman" is person credit. This category is
for "based on / adapted from / from the works of" framing.

No-fire for named franchises or character-anchored sources ("Star Wars
novelizations", "Sherlock Holmes books"), generic media ("comic book
movies"), or source-medium phrases with no named creator.
