# Additional objective notes

## Category Target

Concrete focal subject the movie is about: a real person, event, case,
disaster, war, institution, place-as-subject, or named public story.
Ask: "What subject does the film orbit around?"

## Coverage Decision

- Preferred: Keyword, only when a registry member directly covers the
  subject class or source flag named by the expression.
- Fallback: Semantic plot_events when the expression names a specific
  long-tail subject the registry does not carry.
- Split: use Keyword for a covered generic class and Semantic for the
  named subject only when both are present in this category call.

Do not use broad parent tags as stand-ins for named subjects. If the
ask is "about a specific person," a generic biography tag does not
cover the specific person.

## Boundaries

- Thematic essence -> Story / thematic archetype.
- Mere "has X" presence -> Element / motif presence.
- Time/place setting -> Narrative setting.
- Full plot event sequence -> Plot events.

## No-Fire

No-fire when the expression is abstract, setting-only, mere presence,
or too vague to name a concrete focal subject.
