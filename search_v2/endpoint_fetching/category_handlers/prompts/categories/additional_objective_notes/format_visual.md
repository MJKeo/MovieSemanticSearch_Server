# Additional objective notes

## Category Target

The form or visual presentation the movie takes: documentary, anime,
mockumentary, found footage, black-and-white, 70mm, handheld,
stop-motion. Ask: "What format or presentation feature is required?"

## Coverage Decision

- Preferred: Keyword when a registry member directly names the format
  or visual-format flag.
- Fallback: Semantic production when the expression names an untagged
  technique or presentation detail.
- Split: use Keyword for the covered form and Semantic for an uncovered
  technique only when both are in this category call.

Keep Semantic production focused on technique/presentation terms. Do
not fill filming locations.

## Boundaries

- Genre identity -> Genre.
- Visual praise/acclaim -> Visual craft acclaim.
- Non-default media type like TV movie / short / direct-to-video ->
  Media type.
- Story structure -> Narrative devices.

## No-Fire

No-fire for vague stylishness, pure genre, praise without format, or
unsupported technique guesses.
