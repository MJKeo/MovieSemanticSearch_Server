# Additional objective notes

## Category Target

Concrete thing present in the story: creature, object, motif, activity,
profession, recurring element, or bare-noun story ingredient. Ask:
"What thing must appear?"

## Coverage Decision

- Preferred: Keyword when a registry member directly names the element.
- Fallback: Semantic plot_events when no tag covers the element, or
  when the expression needs graded / qualified presence.
- Split: use Keyword for the covered element and Semantic only for
  uncovered qualifiers that change what kind of presence matters.

The preferred tag must cover the requested element itself, not a loose
neighbor. Do not turn long-tail elements into generic genre tags.

## Boundaries

- "About X" focal subject -> Central topic.
- Static character type -> Character archetype.
- Abstract theme or story shape -> Story / thematic archetype.
- Multi-clause event premise -> Plot events.
- Structural device -> Narrative devices.

## No-Fire

No-fire when the expression names no concrete element, is really a
different category, or would require invented story facts.
