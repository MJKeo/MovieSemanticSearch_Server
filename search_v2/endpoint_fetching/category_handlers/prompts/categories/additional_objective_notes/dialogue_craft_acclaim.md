# Additional objective notes - Dialogue craft acclaim

Search notable dialogue craft: quotable lines, snappy banter, naturalistic
dialogue, rapid-fire patter, monologues, verbal wit, or dialogue praised by
critics/audiences.

Use semantic reception for praise/criticism of dialogue. Use semantic
narrative_techniques when the request is a craft pattern ("rapid-fire
walk-and-talk", "naturalistic overlapping dialogue", "Sorkin-style
banter"). Use both only when both are explicit.

Do not treat named writers as dialogue craft. Literal writer credit belongs
to person credit. Broader narrative-device phrasing without dialogue as the
target belongs elsewhere.

No-fire for generic "well-written" unless dialogue itself is the named
aspect, or for actor/performance praise rather than dialogue craft.
