# Additional objective notes - Character-franchise

## Category question

Does the target name one fictional persona that also anchors a film
franchise or series? If yes, resolve that referent once and let both
retrieval paths consume it.

## Fire

Fire when the named referent is both:

- a character name that can appear in cast/character credits
- the through-line of a character-centered film franchise, series, or
  universe

Character forms are credited persona strings for that referent. Franchise
forms are franchise, series, universe, or umbrella title strings for that
same referent. Keep both form sets narrow to the shared referent.

Use central character framing when the request asks for that character's
movies or story. Use default only for appearance-style wording that still
names a true character-franchise anchor.

## Keep separate

Source-medium and adaptation wording is not part of this payload. Books,
comics, true-story, video-game, and remake-as-source flags belong to
separate category calls.

Structural franchise wording is franchise-side only. Sequel, prequel,
spinoff, reboot, remake-position, subgroup, era, phase, or mainline wording
must not change the character forms.

## No-fire

No-fire when the target is only a character inside someone else's
franchise, only a character-less franchise/universe, a character type, a
motif, an actor, a studio, a genre, or a real biopic subject.
